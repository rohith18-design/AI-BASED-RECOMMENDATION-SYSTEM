HOW IT WORKS

DataModel: Loads user-item ratings from CSV.

Similarity Calculation: Measures similarity between users using Pearson correlation.

Neighborhood Formation: Finds top N (e.g., 2) most similar users.

Recommendation Generation: Suggests items liked by similar users but not yet rated by the target user.
